# Vue-Vux-Demo

> 这是一个基于Vue2实现的webapp项目Demo。


### 技术栈
- 传说中的VUE全家桶(vue vue-router vuex)
- axios
- [Vux](https://vux.li)(一个凑合的 Vue.js 移动端 UI 组件库 )


## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
